Facebox

Facebox detects and recognize faces in photos, and also you can use our labeling app to teach facebox to recognize more new faces.

Uses for Facebox

This capability has a variety of utilities:

  • Detect how many faces are in this photo
  • Recognize those faces automatically by searching your database.
  • Return face ID and other information to you in a json content.

Get started

TThe best way to get started is to check out our APIs, the following steps will guide you how it looks like.

We process the photo when you run facebox that all faces will be marked out, so you just need to use API like this: 

GET:http://facebox-ip/v2/detected_faces.json
RESPONSE:
[ {
      ‘detected’:Boolean,
      ‘style’: ‘front’,
      ‘face_id’:  String,
      ‘recognized’: Boolean,
      ‘labeled_name’: String,
      ‘time_stamp’: Int,
      ‘accuracy’: Int
} … ]
